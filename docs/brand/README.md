# Build Ecosystem brand assets

- `build-ecosystem-mark.svg` — square logomark (constellation of six member
  nodes around a central hub). Use for avatars, favicons, and small
  placements.
- `build-ecosystem-hero.svg` — wide banner used at the top of the project
  README.

Palette: the marks use the Project Telos indigo (`#4636e8`) for the
connecting links, with each member node getting one accent color from the
Build family palette, on a near-black indigo ground (`#160c28`/`#0a0816`).
Assets are SVG so they scale cleanly; export to PNG only if a raster is
required.
